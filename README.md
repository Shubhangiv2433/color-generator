# 🎨 Color Generator App

A simple React app to generate random colors.

## 🚀 Features
- Generate random colors
- Copy color code
- Gradient background support
- Simple and clean UI

## 🛠️ Tech Used
- React JS
- Vite
- JavaScript
- CSS

## ▶️ How to Run
1. Clone the repo
2. Run: npm install
3. Run: npm run dev
